# Music Information Retrieval
{: .no_toc }


> 본 카테고리(Introduction)는 [musicinformationretrieval.com](https://musicinformationretrieval.com)의 내용을 바탕으로 하고 있다.

위 사이트는 음악 정보 검색(MIR)을 위한 교육 자료의 모음으로, 주피터 노트북을 사용하여 Python 코드로 작성되었습니다. [[Github]](https://github.com/stevetjoa/musicinformationretrieval.com)

관리자는 [Steve Tjoa](https://www.stevetjoa.com)라는 구글 엔지니어로, ACM Multimedia, IEEE ICME 컨퍼런스 위원 및 ISMIR 2019 스폰서쉽 의장, CCRMA 연례 워크숍 책임자를 역임했다.
연구 간행물은 [Google Scholar](https://scholar.google.com/citations?user=jYcHr7QAAAAJ)를 참조하세요.
